== Installation ==

This section describes how to install the plugin and get it working.

1. Download the plugin
2. Unzip send-text-message-tool-widget.zip
3. Upload the entire 'send-text-message-tool-widget' folder to your plugins directory
4. Go to the Plugins page in your WordPress Administration area and click to 'Activate' "Send Text Message Tool Widget" on WordPress.
5. Go to Appearance->widgets and add the "Send Text Message Tool Widget" sidebar widget to your sidebar
